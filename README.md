# Alura Books

Este é um projeto desenvolvido para praticar e aprender **JavaScript** e **responsividade**. O projeto consiste em um site de livros que permite aos usuários ordenar os livros com base em diferentes critérios, como preço, disponibilidade e tags.

## Funcionalidades

- **Ordenar por preço**: Os livros podem ser ordenados do menor para o maior preço.
- **Ordenar por quantidade**: É possível listar apenas os livros disponíveis ou todos os livros.
- **Ordenar por tags**: Os livros podem ser filtrados de acordo com diferentes categorias de tags.
- **Responsividade**: O site é responsivo e se adapta a diferentes tamanhos de tela.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Responsividade com Flexbox e Media Queries
